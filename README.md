# C++ Constants Exercises

Raccolta di mini-esercizi focalizzati sull'utilizzo della parola chiave `const` e sulla gestione dei vari tipi di dato in C++.

## Concetti Tecnici Utilizzati

* **Modificatore `const`**: applicato a diversi tipi di dato per garantire l'immutabilità dei valori.
* **Tipi Primitivi**: gestione di `int`, `float`, `double`, `char` e `bool`.
* **Modificatori di Tipo**: utilizzo di `unsigned int` (solo positivi) e `long long int` (grandi numeri a 64-bit).
* **Librerie Standard**:
    * `<iostream>` per l'input/output.
    * `<string>` per la gestione delle stringhe.
    * `<cmath>` per operazioni matematiche avanzate.
* **Strutture di Controllo**: integrazione di costanti all'interno di cicli `for` e istruzioni condizionali `if`.
* **Best Practices**: utilizzo del naming convention in MAIUSCOLO per le costanti e suffissi specifici (`LL`) per i letterali.
